# Predicting-Exams-Using-Pyspark
This project, submitted for the INT315 Cluster Computing course, presents a complete Big Data analytics pipeline for predicting whether a student will pass or fail their final mathematics examination. The solution is built entirely on Apache Spark and PySpark, leveraging distributed in-memory computation to process the UCI Student Performance dataset across all six units of the course syllabus.
The pipeline covers HDFS-based data ingestion (Unit I), RDD transformations and GraphX graph analytics (Unit III), Spark SQL queries with aggregations and window functions (Unit IV), Kafka Structured Streaming simulation (Unit V), and an end-to-end MLlib Machine Learning pipeline featuring StringIndexer, OneHotEncoder, StandardScaler, Logistic Regression, and cross-validated evaluation (Unit VI). The trained classifier achieves over 85% accuracy and 0.87 F1-score on the held-out test set, demonstrating that distributed machine learning at scale is both feasible and practical using the Spark ecosystem.
